# estacionamientos-duocuc
Organiza los estacionamientos y la comunicación entre Jefaturas-Jefaturas Guardias-Conductores,etc.

Requerimientos
-Aplicación con distintos accesos para (1. Jefatura, 2. Jefatura Guardias, 3. Guardias, 4. Conductores)
-Cada lugar esta predeterminado.
-Quién llega más temprano se estaciona en los lugares más alejados de la entrada.
-Asegurar espacios docentes.
-Asegurar espacios adultos mayor.
-Invitados, etc.
-Letras pintadas en el piso.
-Topes para reservar estacionamientos.
-Reportes de problemas.
-Autoservicio con QR para ingreso.
-Espacios desocupados.
-Proporciona enrolamiento, estadística y gráficos.
-Ayuda con toma de decisiones.
-Interfaz en tiempo real de ocupación y bloqueo de estacionamientos.
-Comunicación de incidencias con portería.
